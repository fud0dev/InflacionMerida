# 📈 InflaciónMérida · Mérida XYZ

> **Controlador de precios de la cesta básica en Mérida.**

**InflaciónMérida** es una herramienta de monitorización económica diseñada para seguir la evolución real de los precios en la ciudad. El proyecto automatiza el rastreo de productos de primera necesidad, ofreciendo gráficas históricas y análisis de variaciones para ayudar al ahorro doméstico.

## 🚀 Características Principales

- **Monitor de Precios Diario**: Rastreo automatizado de ~100 productos básicos (alimentación, higiene y hogar).
- **Análisis de Variación**: Indicadores visuales de subidas and bajadas respecto al día anterior.
- **Gráficas Históricas**: Curvas de precios interactivas mediante Chart.js para visualizar tendencias a largo plazo.
- **Categorización Inteligente**: Segmentación de la cesta de la compra para un análisis detallado por grupos.

## 🛠️ Tecnologías Utilizadas

- **Scraper Core**: Python para la extracción diaria de datos directamente de inventarios digitales locales.
- **Frontend Layer**: HTML5, CSS3 con variables de diseño personalizadas y Chart.js para la visualización de datos.
- **Automation Pipeline**: GitHub Actions para la actualización matutina (08:00 AM) de los precios y la reconstrucción del dashboard.

## 📊 Fuentes de Datos

La información se extrae de fuentes de datos reales de establecimientos con presencia en Mérida (p. ej. Mercadona Online), procesando los precios vigentes en el código postal local para garantizar la precisión de la inflación reportada.
